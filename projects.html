Automated Testing Utility (Visa Internship)

In the summer of 2016 I worked for Visa inc. at their headquarteres in Foster City CA. Inefficiency in testing lead me to write an automation script to streamline the E2E testing process. The script was wrritten in Java using Solenium and OpenSSL for authentication of API calls. The utility was implemented in a genereic manner so that it could test multiple internal sites to check for full end 2 end flow. The automation generated a html health report of the particular website.
